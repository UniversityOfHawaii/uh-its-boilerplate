Resources

  • Stay up-to-date: The most up-to-date files for this styleguide are available in GitHub
  • Icons: You can use Font Awesome 5 icons but only the free ones (solids and brands)
  • Bootstrap 4: Read up on Bootstrap 4 Documentation
  • Accessibility: Check your site for accessibility with Site Improve or reserve the SortSite accessibility checker.
    • You can also read up on Accessibility at UH
    • Check your color palette with Accessible color palette builder
  • If you need a simple site: Consider using the UH Web Hosting Service (Option 1) for simple Wordpress sites
  • If you want more customization: Work off of any of the publicly available Wordpress themes for your own customization:
    • UH System (System2018)
    • UH Manoa (Manoa2018)
    • UHM Catalog Child Theme (Manoa2018 theme required)
  • UH Guides: Read up on the UH writing style guide and graphic standards: UH System Writing Style Guide | UH Graphics Standards (print)
